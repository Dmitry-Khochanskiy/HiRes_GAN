# Enhanced-Super-Resolution-GAN network in torch
An implemintation of Enhanced-Super-Resolution GAN from #https://github.com/AquibPy/Enhanced-Super-Resolution-GAN . It was tested on WIN 10 with Nvidia GeForce GTC 1660 and RTX 3050.

# Prerequisites
Anaconda, CUDA
Env can be loaded from pytorchenv.yml

# Usage
It can run from notebooks

# Data set for training
[Colorectal Histology MNIST (https://www.kaggle.com/kmader/colorectal-histology-mnist)]
